# netflix Clone

Un clon pixel-perfect de la página principal de Netflix construido con **HTML5, CSS3 y JavaScript puro** — sin frameworks ni dependencias.

![Netflix Clone Preview](https://assets.nflxext.com/ffe/siteui/acquisition/ourStory/fuji/desktop/hero-atf.jpg)

## Características

- ✅ Navbar con efecto de scroll (transparente → negro)
- ✅ Hero section con fondo cinematográfico y formulario de email con validación
- ✅ 4 secciones de características con videos animados de Netflix
- ✅ Animación de tarjeta de descarga flotante
- ✅ Sección FAQ con acordeón interactivo
- ✅ Footer completo con todos los links
- ✅ Scroll reveal (animaciones al hacer scroll)
- ✅ Toast notifications
- ✅ 100% responsivo (mobile, tablet, desktop)
- ✅ Sin dependencias externas



## Cómo ejecutar

### Opción 1 — Abrir directamente

```bash
# Clona el repositorio
git clone https://github.com/tu-usuario/netflix-clone.git

# Abre index.html en tu navegador
open netflix-clone/index.html
```

### Opción 2 — Con servidor local

```bash
# Con Python
python3 -m http.server 8080

# Con Node.js (npx)
npx serve .

# Luego abre http://localhost:8080
```

## 🛠 Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura semántica |
| CSS3 | Variables, Grid, Flexbox, Animaciones |
| JavaScript ES6+ | FAQ, validación, scroll effects |
| Google Fonts | Tipografía |
| Intersection Observer API | Scroll reveal |

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 768px` | Desktop — dos columnas por sección |
| `≤ 768px` | Tablet/Mobile — una columna, stacked |
| `≤ 480px` | Mobile pequeño — selector de idioma oculto |
